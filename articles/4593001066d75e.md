Railsでenumを用いたステータス管理を行っており、相互変換を行いたい。
## モデル
```ruby:user.rb
class User < ApplicationRecord
  # 以下ステータス
  enum status: { public: 1, private: 2, banned: 3 }
end
```
## キーから値を得る
```ruby
User.statuses[:public]
#=> 1
```

## 値からキーを得る
```ruby
User.statuses.invert[1]
#=> "public"
```
